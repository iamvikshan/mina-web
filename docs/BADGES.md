### **How Ranks are Calculated**

```typescript
// Example from guardian-ranks.ts
const managedGuilds = discordGuilds.filter((guild) => {
  const permissions = BigInt(guild.permissions);
  return (permissions & 0x8n) === 0x8n; // ADMINISTRATOR permission
});

const serverCount = managedGuilds.length;
const guardianRank = getGuardianRank(serverCount);
```

### **How Achievements are Checked**

```typescript
// Example from achievements.ts
const achievements = checkAchievements(
  serverCount, // Number of managed realms
  configuredCount, // Number of configured realms
  userData // User data including login time
);
```

1. **Dashboard Welcome Card**: Shows current rank with badge
2. **Guardian Badge Component**: Displays rank with progress bar
3. **Recent Achievements Section**: Shows last 4 earned achievements
4. **User Avatar Dropdown**: Shows rank in header (e.g., "Lv2 Scout")
